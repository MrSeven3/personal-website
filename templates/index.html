my homelab

totally overkill lmao. i run voidauth for sso, and have like 12 different compose stacks running stuff. the docker containers are managed by komodo (great software). everything with a public domain is through nginx (finally out of proxyauth hell). i run my own tangled.org knot and spindle, and i run game servers with pterodactyl. theres a public uptime monitor here.

i also run a very glitchy home assistant that is more often broken than functional. i have a grafana dashboard because i can. all of this runs on a raspberry pi 5 with 8gb of ram, a pi 3b with 1gb, an old gaming laptop with 16gb, a free oracle vps with 24gb, and a small server provided by hack club (nest is a great resource if you're in hack club)
